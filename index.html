My Projects

image similarity recognition

This project uses KD tree data structure to create a similar color image recommendation system. Rather than recommending images based on content, our system focuses on color similarity. Users can select an image, and the program returns visually similar color images, including the query image itself. We curated a dataset of 226 images from various sources, ensuring diverse color representation. Using KD trees, we derived essential metrics like hue, saturation, and value (lightness/darkness) of each image to deliver precise matches based on user queries. Additionally, the program integrates functionalities for users to insert, delete, and retrieve similar images using nearest neighbor search, providing a dynamic and interactive experience. To enhance accessibility, we designed a user-friendly GUI using the TKINTER library..
